# 批量图片审核平台

一个功能完整的批量图片审核平台，支持CSV导入、图片预览、筛选、分页、批量审核和结果导出。

## 功能特性

- ✅ **CSV导入**：支持导入CSV文件，自动识别图片URL列
- ✅ **图片预览**：自动预加载图片，平铺展示，支持3:4比例显示
- ✅ **图片元数据识别**：自动识别图片的宽度、高度、宽高比和文件大小
- ✅ **字段展示**：每张图片下方显示该行数据的其他字段信息（包括图片尺寸信息）
- ✅ **筛选功能**：顶部可根据CSV表头进行多条件筛选
- ✅ **尺寸筛选**：支持按图片宽度、高度、宽高比、文件大小进行范围筛选
- ✅ **分页功能**：支持每页显示10/50/100张图片
- ✅ **批量操作**：支持全选本页、批量通过选中图片
- ✅ **单个审核**：支持对单张图片进行通过/不通过操作，可输入不通过理由
- ✅ **结果导出**：导出审核结果CSV，包含审核结果和不通过理由两列

## 使用方法

1. **打开平台**
   - 直接在浏览器中打开 `index.html` 文件

2. **导入CSV**
   - 点击"导入CSV"按钮
   - 选择包含图片URL的CSV文件
   - 系统会自动识别图片URL列（支持列名：角色图片（超链接）、抠图url、图片url等）

3. **审核图片**
   - 使用顶部筛选框筛选需要审核的数据
   - **图片尺寸筛选**：支持按宽度、高度、宽高比、文件大小筛选
     - 例如：宽度输入 `>1000` 筛选宽度大于1000px的图片
     - 例如：高度输入 `500-2000` 筛选高度在500-2000px之间的图片
     - 例如：宽高比输入 `>1.5` 筛选横向图片
     - 例如：文件大小输入 `<500` 筛选小于500KB的图片
   - 选择每页显示数量（10/50/100）
   - 点击图片上的"通过"或"不通过"按钮进行审核
   - 不通过时需要输入理由
   - 可以勾选多张图片后点击"批量通过选中"进行批量审核

4. **导出结果**
   - 完成审核后，点击"导出审核结果"按钮
   - 导出的CSV文件会在原数据基础上增加两列：
     - `审核结果`：通过/不通过/待审核
     - `不通过理由`：不通过时输入的理由

## 图片尺寸筛选语法

新增的图片尺寸筛选功能支持以下语法：

### 基本语法
- `>数值` - 大于指定值（例：`>1000`）
- `<数值` - 小于指定值（例：`<500`）
- `>=数值` - 大于等于指定值（例：`>=800`）
- `<=数值` - 小于等于指定值（例：`<=1920`）
- `数值` - 等于指定值（例：`1920`）
- `数值1-数值2` - 在范围内（例：`500-2000`）

### 筛选示例

**宽度筛选（单位：px）**
- `>1000` - 宽度大于1000像素
- `<800` - 宽度小于800像素
- `1080-1920` - 宽度在1080到1920像素之间

**高度筛选（单位：px）**
- `>1200` - 高度大于1200像素
- `<=1080` - 高度小于等于1080像素
- `600-1200` - 高度在600到1200像素之间

**宽高比筛选**
- `>1.5` - 宽高比大于1.5（横向图片）
- `<1` - 宽高比小于1（纵向图片）
- `0.8-1.2` - 宽高比接近正方形
- `1.77` - 宽高比约等于16:9

**文件大小筛选（单位：KB）**
- `<500` - 文件小于500KB
- `>1000` - 文件大于1MB（1000KB）
- `100-500` - 文件在100KB到500KB之间

## 技术说明

- 纯前端实现，无需服务器
- 使用原生JavaScript，无需依赖
- 支持CSV格式解析（处理引号、逗号等特殊情况）
- 自动识别图片URL列
- 自动获取图片元数据（宽度、高度、宽高比、文件大小）
- 元数据缓存机制，避免重复请求
- 支持范围筛选语法（>、<、>=、<=、范围）
- 图片预加载优化体验
- 响应式设计，支持不同屏幕尺寸

## 注意事项

- CSV文件需要包含图片URL列
- 图片URL可以是完整URL或相对路径
- 图片元数据（宽度、高度、文件大小）会在图片加载后自动识别
- 如果图片跨域，可能无法获取文件大小信息（需要服务器支持CORS）
- 审核状态会保存在内存中，刷新页面会丢失，请及时导出结果
- 图片元数据会缓存在内存中，提高筛选性能
- 建议使用Chrome、Firefox、Edge等现代浏览器
